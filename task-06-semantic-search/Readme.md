# Task 06 - Semantic Search System

## Overview

This project implements a semantic search system for CoreTech.

The system uses TF-IDF Vectorization and Cosine Similarity to find the most relevant information from a knowledge base dataset.

## Features

- Knowledge base dataset with 30+ records
- Service descriptions
- FAQs
- Company information
- TF-IDF vectorization
- Cosine similarity matching
- Top 3 matching results
- Similarity score display

## Files

### coretech_knowledge_base.csv
Contains service descriptions FAQs and company information.

### semantic_search.py
Implements semantic search using TF-IDF and cosine similarity.

## Technologies Used

- Python
- Pandas
- Scikit-learn

ai chatbot

## Output

The system returns the top 3 matching records along with similarity scores.
