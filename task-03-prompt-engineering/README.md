# Task 03 - Prompt Engineering

## Overview

This task contains prompt engineering examples for AI and business-related use cases.

The prompts include:

* Website Content Prompts
* Social Media Prompts
* Client Email Prompts
* Business Idea Prompts
* Chatbot Response Prompts

A total of 20 prompts were created for this task.


## Objective

The objective of this task is to practice prompt engineering and understand how prompts can be improved for better AI responses.
