# 🤖 CoreTech AI Assistant (Task 05 - Streamlit Chatbot)

## Overview

This project is an upgraded version of Task 04 chatbot, converted into a **Streamlit web application**.
The chatbot uses a FAQ dataset and rule-based keyword matching to respond to user queries about CoreTech services, including AI, web development, mobile apps, cloud, and automation.
It provides a simple, interactive, and professional web interface.

## Project Evolution

 **Task 04** Console-based rule-based chatbot using Python
 **Task 05** Web-based chatbot using Streamlit with UI improvements

##  Features

*  FAQ dataset with 25+ questions and answers
* Keyword-based intelligent matching system
*  Interactive chat-style input box
*  Unknown question handling with friendly message
*  Streamlit web interface (professional UI)
*  CoreTech branding integration
*  Clean and responsive layout
*  Continuous user interaction support

## Technologies Used

* Python
* Streamlit
* Pandas
* CSV Dataset

## How to Run

### Step 1: Install dependencies

bash
pip install streamlit

### Step 2: Run the application

bash
streamlit run app.py


## Example Queries

- mobile app

- cloud
- contact
